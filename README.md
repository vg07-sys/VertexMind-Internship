# 📖 About This Repository

This repository contains all the assignments and project deliverables completed during the VertexMind Digital Marketing Internship.

The project revolves around **FicTabs**, a fictional streaming platform designed for drama enthusiasts, offering multilingual subtitles, fewer ads, and global accessibility. Through this case study, I explored audience research, content strategy, social media planning, and paid advertising while building a complete digital marketing portfolio.
